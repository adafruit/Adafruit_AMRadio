void Adafruit_AMRadio::write ( uint16_t  center_frequency)

write(n) is equivalent to analogWrite(n); accepts 10-bit input (0-1023). Any function that uses DAC for audio (WAV player, Talkie, etc.) can be easily modified to use this instead of the DAC.
